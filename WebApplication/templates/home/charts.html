Project Introduction

Project Title
배봉지(과일봉지) 숫자 카운팅 AI모델 개발
AI Model Development for Counting Fruit Bags (Pear Bags)
Project Description
In pear orchards, fruit bags are used to improve pear quality. Every year, in collaboration with disaster insurance companies, the number of fruit bags is counted in spring and after natural disasters (e.g., typhoons) to assess the level of damage. This project aims to address this issue by developing a vision AI model that can automatically count fruit bags from images and videos collected through cameras and eliminating the need for manual counting. Also , this could reduce manual labor costs and improve accuracy in agriculture and disaster insurance by automating fruit bag counting Input : Provided images (videos) which contains fruit bags. Images may show the same fruit bag multiple times due to different shooting angles Output : The Counted and detected fruit bags on image . Functions : Develop a vision AI model that must detect and count the fruit bags without duplication
